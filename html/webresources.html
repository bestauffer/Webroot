Web Resources

  1. W3Schools
    W3 Schools is a website with many different beginner and more advanced tutorials. This link specifically takes you to a section on Full Stack.
    Rating: 5(high)

  2. freeCodeCamp
    freeCodeCamp is a website with walkthrough tutorials for different area of programming such as responsive web design & front end development libraries. Each certification is about 300 hours of work plus it has thousands of hours of interview prep.
    Rating: 3

  3. W3C: HTML
    You can use this validator to check the accuracy of your HTML code. It will let you know if there's any missing semicolons or brackets. This will be useful because it provides reassurance that you are coding properly.
    Rating: 4

  4. W3C: CSS
    This does the same thing as the validator above but instead it works with CSS.
    Rating: 4

  5. GitHub
    GitHub provides a cloud based service for someone to be able to manage their code online and is amazing for team projects. It allows for easy version control as well as hosting capabilities.
    Rating: 4
